MEDIA QUERIES!

Le media queries sono degli strumenti che ci permettono di limitare una porzione di CSS solamente a schermi/dispositivi che rispettano determinate condizioni. Ad esempio, è possibile creare una porzione di selettore CSS che verranno applicati con le loro regole solamente su uno schermo più grande di tot px sull'asse orizzontale, oppure possiamo trasformare/nascondere determinate porzioni della pagina ad una stampante o ad uno screen reader.
